    Features

    Everything a real team does,
    automated end to end

    ⚡
    Real-time agent collaboration
    Watch agents debate architecture, review each other's code, and resolve blockers live in the dashboard. Powered by Redis Streams — every message is persisted and auditable.
    Redis Streams Live SSE Feed Persistent Audit Log
    # Agents communicate via Redis inboxes await redis.xadd( "agent:senior_dev_1:inbox", {"from": "eng_manager", "task": "implement /auth/login", "priority": "high"} ) # Agents reply, review, iterate...
    🔒
    Sandboxed code execution
    All code runs in an isolated Docker sandbox before being pushed. Tests must pass — no exceptions. The CI result is published back to the agent feed.
    Docker sandbox Auto-CI
    🧠
    Persistent agent memory
    Each agent writes lessons learned to PostgreSQL. The next time you submit a task, agents build on prior experience — no repetition, continuous improvement.
    PostgreSQL Cross-task learning
    ⎇
    Automatic GitHub push
    Agents commit code to a new repo on your GitHub account with a real, readable commit history — architecture docs, implementation, tests, and README in separate commits.
    🤖
    Works with GitHub Copilot
    No Anthropic API key required. If you have a GitHub Copilot subscription, agents auto-detect it and use the Copilot backend — same Claude models, zero extra cost.
    GitHub Copilot Anthropic API
    📊
    Live engineering dashboard
    Watch the full agent message feed, kanban board, CI results, token usage, and team wiki — all updating in real time at /dashboard. Stakeholders get a high-level view at /report.
    💬
    Human-in-the-loop
    When agents are stuck or need a decision, they surface a question to you through the dashboard. You reply, they continue — keeping you in control without blocking progress.
    🤖
    Grow the team at runtime
    Click "+ Add Agent" in the dashboard. Describe the role. Claude generates the config files, a Docker container spins up from the pre-built base image, and the new specialist joins the team — in under 30 seconds. No files to write.
    Claude generates config Docker SDK <30 seconds
    🐳
    One command deploy
    The entire team — 12 Docker containers including Redis, PostgreSQL, sandbox, and 8+ agents — starts with a single command. No cloud accounts needed.
    docker compose up

    Quick Start

    Up and running in minutes

    1
    Clone the repo
    git clone https://github.com/RajuRoopani/team-claw cd team-claw
    2
    Configure your .env
    Add your GitHub token and username. No Anthropic key needed if you have GitHub Copilot.
    GITHUB_TOKEN=ghp_... GITHUB_USERNAME=your-username
    3
    Launch the team
    docker compose up --build -d
    4
    Submit your first task
    curl -X POST http://localhost:8080/task \ -H "Content-Type: application/json" \ -d '{"title":"Build a REST API for a blog"}'
